# Hospital Care Management System

A robust, modern Desktop Application built using **JavaFX** and **Maven** designed to streamline clinic workflows, billing cycles, patient medical records, and appointment schedules.

## 🚀 Features
- **Role-Based Authentication:** Clean panels for Admin, Doctors, and Patients.
- **Appointment Management:** Real-time scheduling with cascading logic to ensure zero orphaned data.
- **Automated Billing System:** Dynamic invoice generation.
- **Data Safety:** Atomic write flushing back to disk files to ensure zero data corruption.

## 🛠️ Built With
- **Language:** Java 11
- **UI Framework:** JavaFX 13 (with FXML and custom CSS styling)
- **Build Tool:** Maven
